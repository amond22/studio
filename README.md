# EduScan Attend - Balmiki Lincoln College

This is a modern, AI-powered Student Attendance Management System.

## ⚡ Quick Start (Run in VS Code)

1. **Open Terminal**: Press `Ctrl + ~` (or `Cmd + ~` on Mac).
2. **Install Dependencies**: Type `npm install` and press Enter.
3. **Run the App**: Type `npm run dev` and press Enter.
4. **Open in Browser**: Go to `http://localhost:9002`.

*Note: If you have the "Debugger for Chrome" extension, you can also just press **F5** in VS Code to start.*

## 🔑 Default Credentials

- **Admin Portal**: ID: `admin` | PW: `admin-password`
- **Teacher Portal**: ID: `teacher` | PW: `teacher-password`
- **Student Portal**: ID: `student` | PW: `student-password`

## 🛠 Features
- **Persistent Data**: Everything you save stays in your browser's memory.
- **QR Scanning**: Real-time attendance via mobile camera.
- **Admin Controls**: Manage users, subjects, and college branding.
- **Mobile Ready**: Fully responsive design for students on the go.

## 📦 Deployment
When you are ready to go live, this project is compatible with **Firebase App Hosting**. Simply push to GitHub and connect your repository.
